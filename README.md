# QuickMove Website Landing Page

Files included:
- index.html — QuickMove competition landing page
- quickmove_updated_mvp_prototype.html — embedded clickable MVP prototype

How to use:
1. Open index.html in a browser.
2. The prototype is embedded near the bottom and also opens in a new tab.
3. Upload both files together to your website host so the embedded prototype works.

Suggested meta description:
QuickMove is a privacy-first contact-before-tow platform that helps residents and property managers resolve private parking conflicts through verified vehicle registration, masked alerts, and documented escalation.
